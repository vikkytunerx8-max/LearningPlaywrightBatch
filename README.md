## Chapters

### Chapter 1: Basics
Introduction to fundamental Playwright concepts and JavaScript basics:
- `01_basic.js` - Basic setup and getting started
- `02_JS_Step_By_Step.js` - Step-by-step JavaScript guide
- `03_verify_setup.js` - Verification of development setup
- `04_hot_code.js` - Hot code reloading examples

### Chapter 2: Core JavaScript Concepts
Deep dive into JavaScript fundamentals:
- `05_Core_Comments_JS.js` - Understanding comments in JavaScript
- `06_Core_Identifier_JS.js` - Working with identifiers
- `07_var_let_const.js` - Variable declarations and scoping

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm (Node Package Manager)
- Playwright

### Installation

1. Clone the repository:
```bash
git clone https://github.com/vikkytunerx8-max/LearningPlaywrightBatch.git
cd LearningPlaywrightBatch
```

2. Install dependencies:
```bash
npm install
```

3. Install Playwright browsers:
```bash
npx playwright install
```

## Usage

Run individual test files:
```bash
node chapter_01_Basics/01_basic.js
node chapter_02_Java_Concepts/05_Core_Comments_JS.js
```

## Topics Covered
